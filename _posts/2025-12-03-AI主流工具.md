场景五：Canvas

Gemini Canvas 是 Google 推出的一个专为协作创作和编程设计的全新界面。

适合写作和跑代码，还可以生成PPT

| **任务类型**   | **使用标准 Gemini**          | **使用 Gemini Canvas**       |
| -------------- | ---------------------------- | ---------------------------- |
| **查询信息**   | ✅ 更好（如：查天气、找事实） | ❌ 不推荐                     |
| **简单问答**   | ✅ 更好                       | ❌ 不推荐                     |
| **编写长代码** | ❌ 容易中断，难以调试         | ✅ **推荐**（易于迭代修改）   |
| **撰写长文章** | ❌ 格式调整麻烦               | ✅ **推荐**（易于排版润色）   |
| **数据分析**   | ⚠️ 可以，但结果分散           | ✅ **推荐**（代码与图表同屏） |

## 04 Cursor

官网，[链接](https://cursor.com/cn/home?from=agents)

类似的软件Google Antigravit，Windsurf (by Codeium)，Trae (字节跳动/ByteDance)，VS Code + "Agent 级" 插件 (不换 IDE 的方案)...

> 如果你想**写**代码，用 **Cursor**；如果你想**完成任务**（包含写代码、跑命令、Git操作），用 **Claude Code**。
>
> Cursor - 你是主驾驶，AI 是副驾驶。你依然需要对每一行代码负责，你的手依然在键盘上。
>
> Claude Code - 你是老板，它是员工。你关注的是“结果”（测试过了吗？功能实现没有？），而不是它中间怎么敲的代码。

- 代码补齐
- 代码预测
- Agent模式
- MCP
